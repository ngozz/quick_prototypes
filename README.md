# Quick Prototypes

This repository contains a collection of quick prototype web games designed to demonstrate and explore various game ideas and mechanics.

## Contents

* **Lantern:** A color mixing light puzzle game where you place lanterns, bend beams, and mix colors to hit targets (`lantern/lantern_color_mix_puzzle.html`).
* **Various Prototypes:** A collection of other game prototypes including chain reactions, timing games, and puzzle mechanics (found in the `20260410/` directory).

## How to Play

Since these are straightforward web-based prototypes, you can simply open the HTML files directly in your web browser to play them. No server or build step is required. You can also open the github pages hosted website.
